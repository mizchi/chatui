# Voice Chat UI

## Features

- Voice Activity Detection (VAD) by vad.js
- Speech to Text by OpneAI Whisper
- Text Generation by OpenAI or AnthropicAI
- Text to Speech by Voicevox

## Requirements

- OpenAI
- Voicevox
- (Optional) AnthropicAI

## Develop

```bash
$ pnpm install
$ npx zx copy.mjs
$ pnpm vite dev
```
